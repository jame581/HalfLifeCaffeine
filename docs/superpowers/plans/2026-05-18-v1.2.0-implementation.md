# HalfLife Caffeine v1.2.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v1.2.0 of HalfLife Caffeine — let users edit the time of, or delete, a drink they've already logged today, and keep the phone companion's `drinkHistory` mirror consistent.

**Architecture:** Six layers from the data outward. (1) `CaffeineModel` gets pure mutation methods (`removeDoseAt`, `adjustDoseTime`, `getTodayLogIndices`) with unit-test coverage. (2) `SyncManager` switches from incremental delta sync to full-day "replace-day" resyncs. (3) The companion HTML gains a `replace-day` branch in `onWatchData`. (4) `LogView` grows a cursor (UP/DOWN selection + highlight), mirroring `HistoryView`'s pattern. (5) Two new `Menu2InputDelegate` files handle the action and time-offset menus. (6) `HalfLifeCaffeineApp` orchestrates with new `removeDose` and `adjustDoseTime` methods, including dual-day sync on cross-midnight edits.

**Tech Stack:** Monkey C, Connect IQ SDK 9.1.0, Toybox APIs (`Application`, `Graphics`, `WatchUi`, `Communications`, `Time`). JavaScript/HTML for the companion page.

**Spec:** `docs/superpowers/specs/2026-05-18-v1.2.0-edit-delete-design.md`

---

## Conventions used throughout this plan

- **Type annotations** on function signatures are omitted per `CLAUDE.md` (SDK 9.1.0 produces an error storm when they are present). Resulting "Cannot determine container access" warnings are cosmetic and expected.
- **Drawing coordinates use integer math.** Floats crash `drawText`/`fillRectangle` on-device (e.g. `height * 18 / 100` not `height * 0.18`).
- **Storage round-trip convention:** dicts do NOT survive `Application.Storage`. Persist as arrays of primitives. (Not touched in this release, but keep this in mind if storage formats change.)
- **Tests** are `(:test)`-annotated functions in `test/` that return a boolean (true = pass). Run via VS Code "Run Tests" launch config — prompts for a simulator device.
- **Commits:** small, frequent, per-task. Use `git add <specific-file>` — never `git add -A`.

```monkey-c
    // The result is clamped to <= nowEpoch (no future doses). Negative offsets crossing
    // midnight are allowed (caller re-evaluates day-membership).
    // Returns the resulting :time, or -1 if the index is out of bounds.
    function adjustDoseTime(doseIndex, offsetSeconds, nowEpoch) {
        if (doseIndex < 0 || doseIndex >= _doses.size()) { return -1; }
        var newTime = _doses[doseIndex][:time] + offsetSeconds;

        return newTime;
    }
```

```monkey-c
    // Return _doses indices for doses falling on today's local day, sorted by :time ascending.
    // Used by LogView to translate cursor position → master _doses index safely.
    function getTodayLogIndices(nowEpoch) {
        var todayStart = getMidnightEpoch(nowEpoch);
        // Collect [index, time] pairs for today's doses.
        var pairs = [];
        for (var i = 0; i < _doses.size(); i++) {
            if (_doses[i][:time] >= todayStart) {
                pairs.add([i, _doses[i][:time]]);
            }
        }
        // Insertion sort by time (n is small — ~10 doses/day worst case).
        for (var i = 1; i < pairs.size(); i++) {
            var cur = pairs[i];
            var j = i - 1;
            while (j >= 0 && pairs[j][1] > cur[1]) {
                pairs[j + 1] = pairs[j];
                j -= 1;
            }
            pairs[j + 1] = cur;
        }
        var indices = [];
        for (var i = 0; i < pairs.size(); i++) {
            indices.add(pairs[i][0]);
        }
        return indices;
    }
```

```javascript
function onWatchData(data) {
    if (!data || data.type !== 'drinks' || !data.data) { return; }
    if (data.mode === 'replace-day' && typeof data.ymd === 'number') {
        // Drop existing entries whose local date matches the given ymd, then append.
        drinkHistory = drinkHistory.filter(function(d) {
            return localYmd(d.time) !== data.ymd;
        });
        for (var i = 0; i < data.data.length; i++) {
            drinkHistory.push(data.data[i]);
        }
    } else {
        // Legacy / append mode — preserve existing behavior for older watch versions.
        for (var i = 0; i < data.data.length; i++) {
            drinkHistory.push(data.data[i]);
        }
    }
    localStorage.setItem('drinkHistory', JSON.stringify(drinkHistory));
}

function localYmd(epochSeconds) {
    var d = new Date(epochSeconds * 1000);
    return d.getFullYear() * 10000 + (d.getMonth() + 1) * 100 + d.getDate();
}
```

```monkey-c
import Toybox.WatchUi;
import Toybox.Graphics;
import Toybox.Application;
import Toybox.Time;

class LogView extends WatchUi.View {

    // Index into the rendered (newest-first) list. Default 0 = most recent dose.
    var selectedIndex;

    // Cached array of _doses indices in rendered (newest-first) order.
    // Rebuilt every onUpdate from caffeineModel.getTodayLogIndices.
    var dayIndices;

    // Cached array of dose dicts in rendered (newest-first) order, parallel to dayIndices.
    var dayDoses;

    function initialize() {
        View.initialize();
        selectedIndex = 0;
        dayIndices = [];
        dayDoses = [];
    }

    function onUpdate(dc) {
        var app = Application.getApp();
        var now = Time.now().value();
        var width = dc.getWidth();
        var height = dc.getHeight();

        dc.setColor(Colors.BG, Colors.BG);
        dc.clear();

        dc.setColor(Colors.TEXT_SECONDARY, Graphics.COLOR_TRANSPARENT);
        dc.drawText(width / 2, height * 18 / 100, Graphics.FONT_XTINY,
            "Today's Drinks", Graphics.TEXT_JUSTIFY_CENTER | Graphics.TEXT_JUSTIFY_VCENTER);

        if (app.caffeineModel == null) { return; }

        // Build newest-first caches from chronological-ascending model indices.
        var ascIndices = app.caffeineModel.getTodayLogIndices(now);
        var doses = app.caffeineModel.getDoses();
        dayIndices = [];
        dayDoses = [];
        for (var i = ascIndices.size() - 1; i >= 0; i--) {
            dayIndices.add(ascIndices[i]);
            dayDoses.add(doses[ascIndices[i]]);
        }

        if (dayDoses.size() == 0) {
            selectedIndex = 0;
            dc.setColor(Colors.TEXT_DIM, Graphics.COLOR_TRANSPARENT);
            dc.drawText(width / 2, height / 2, Graphics.FONT_SMALL,
                "No drinks today", Graphics.TEXT_JUSTIFY_CENTER | Graphics.TEXT_JUSTIFY_VCENTER);
            return;
        }

        // Re-clamp selectedIndex if mutations shrank the list.
        if (selectedIndex >= dayDoses.size()) {
            selectedIndex = dayDoses.size() - 1;
        }
        if (selectedIndex < 0) { selectedIndex = 0; }

        var listTop = height * 28 / 100;
        var listBottom = height * 82 / 100;
        var listHeight = listBottom - listTop;
        var lineHeight = height * 10 / 100;
        var maxVisible = listHeight / lineHeight;

        var rowInsetLeft = width * 10 / 100;
        var rowInsetRight = width * 10 / 100;
        var rowWidth = width - rowInsetLeft - rowInsetRight;

        for (var i = 0; i < dayDoses.size() && i < maxVisible; i++) {
            var dose = dayDoses[i];
            var y = listTop + (i * lineHeight);
            var isSelected = (i == selectedIndex);

            if (isSelected) {
                dc.setColor(Colors.TRACK, Colors.BG);

            dc.setColor(Colors.TEXT_PRIMARY, Graphics.COLOR_TRANSPARENT);
            dc.drawText(width / 2, y, Graphics.FONT_XTINY,
                line, Graphics.TEXT_JUSTIFY_CENTER | Graphics.TEXT_JUSTIFY_VCENTER);
        }
    }

    // Move the cursor by delta rows (clamped). Called by LogDelegate on UP/DOWN.
    function moveSelection(delta) {
        var newIndex = selectedIndex + delta;
        if (newIndex < 0) { newIndex = 0; }
        if (newIndex >= dayDoses.size()) { newIndex = dayDoses.size() - 1; }
        if (newIndex != selectedIndex) {
            selectedIndex = newIndex;
            WatchUi.requestUpdate();
        }
    }

    // Return the master _doses index for the currently-selected row, or -1 if empty.
    function getSelectedDoseIndex() {
        if (dayDoses.size() == 0) { return -1; }
        if (selectedIndex < 0 || selectedIndex >= dayIndices.size()) { return -1; }
        return dayIndices[selectedIndex];
    }
}
```

## Rollback

If a regression appears after release:

1. **Watch-side regression:** revert the offending commit(s) on a hotfix branch, rebuild `.iq` with `monkeyc -e -f monkey.jungle -y D:/Garmin/Key/developer_key -o bin/HalfLifeCaffeine.iq`, upload as `1.2.1`.
2. **Companion-side regression:** on the `gh-pages` branch, revert `settings/index.html` to the v1.1.0 version (`git show v1.1.0:companion/settings/index.html > settings/index.html` is close but `gh-pages` has its own path layout — copy from main's last-good commit instead). Push; Pages redeploys within ~90s. No `.iq` rebuild needed.
3. **Sync protocol drift** (rare — would manifest as duplicate entries in companion History tab): the protocol is backwards-compatible by design (missing `mode` field = legacy append), so the only way this breaks is a malformed `mode: "replace-day"` message. Audit `SyncManager.syncDayToPhone` for unexpected `ymd` values.

The model-layer commits (Section A) are independently testable and isolated — they can be reverted without affecting Sections B+. The companion-side change (Section C / Task F2) is the only change that affects users on older watch versions, and it's strictly additive (existing append behavior preserved).
